### Exemplo de prompt (Zero-Shot)
Caso de uso: quando a tarefa e direta e o modelo nao precisa de exemplos previos para entender o que deve fazer. Aqui, o objetivo e extrair campos basicos de um CSV anexado da forma mais simples possivel.

Entrada:
```code-block
Extraia os dados do CSV e retorne nome, sobrenome e matricula.

Agora processe esta entrada do arquivo anexado.
```

### Diagrama de Fluxo

```mermaid
flowchart TD
    A[Entrada: nova tarefa ou dado bruto] --> B[Prompt: instrucao direta sem exemplos]
    B --> C[Interpretar objetivo e formato esperado]
    C --> D[Aplicar conhecimento interno diretamente ao caso]
    D --> E[Saida final: resposta direta]

    classDef input fill:#2563eb,stroke:#1e3a8a,color:#fff
    classDef prompt fill:#475569,stroke:#334155,color:#fff
    classDef technique fill:#d97706,stroke:#92400e,color:#fff
    classDef reasoning fill:#7c3aed,stroke:#5b21b6,color:#fff
    classDef action fill:#0f766e,stroke:#115e59,color:#fff
    classDef observation fill:#0891b2,stroke:#0e7490,color:#fff
    classDef decision fill:#dc2626,stroke:#991b1b,color:#fff
    classDef output fill:#16a34a,stroke:#166534,color:#fff

    class A input
    class B prompt
    class C reasoning
    class D action
    class E output
```

> **Caracteristica:** Nenhum exemplo e fornecido. O modelo depende exclusivamente do conhecimento pre-treinado para interpretar e executar a tarefa.
